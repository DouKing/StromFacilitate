## StromFacilitate

> 封装了一些 iOS 开发常用的处理

## 使用 `Cocoapods` 安装

将下面的内容添加进 `Podfile`，并运行 `pod install`：

```ruby
pod 'StromFacilitate'
```

或

```ruby
pod 'StromFacilitate/Core'
pod 'StromFacilitate/UI'
pod 'StromFacilitate/Safe'
pod 'StromFacilitate/Network'
```
